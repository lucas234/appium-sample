# appium-sample
1.Java
- appium client version（=5.0.4），maven pom.xml：
    ```xml
    <dependency>
        <groupId>io.appium</groupId>
        <artifactId>java-client</artifactId>
        <version>5.0.4</version>
    </dependency>
    ```
    
2.Python
- appium client version（=0.46）
    ```
    pip install -U Appium-Python-Client
    ```
